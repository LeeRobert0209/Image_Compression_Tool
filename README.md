# 图片极限压缩工具使用指南 (User Guide)
**工具版本**: v1.0
**作者**: Antigravity

本工具旨在为您提供简单、美观且强大的图片批量压缩体验。支持 JPG、PNG、WebP 等常见格式。

## 🚀 快速开始

1.  进入本工具所在的文件夹。
2.  双击运行 **`启动压缩工具.bat`**。
    - *(首次运行时会自动安装必要组件，可能需要几秒钟)*

## ✨ 主要功能

### 1. 拖拽操作
- 您可以直接将 **图片文件** 或 **文件夹** 拖入软件中间的虚线区域。
- 软件会自动扫描并识别所有支持的图片。

### 2. 两种压缩模式
- **智能模式 (默认)**
    - **原理**: 您指定一个目标大小（如 150KB），软件会自动尝试调整画质，尽最大努力将图片压缩到该大小以内，同时保持最高画质。
    - **适用**: 对文件大小有严格限制的场景（如上传证件照、网页优化）。
- **固定质量模式**
    - **原理**: 您指定一个固定的压缩比例/画质（如 80%），软件将统一以该比例处理所有图片。
    - **适用**: 只需要适度减小体积，不需要严格控制具体大小的场景。

### 3. 高级选项
- **限制最大宽度**: 
    - 勾选后，您可以限制图片的最大宽度（如 1080px）。如果原图超过此宽度，将会等比缩小。这是减小体积最有效的方法。
- **转换为 WebP**: 
    - 勾选后，所有图片将被转换为 Google 开发的 WebP 格式。在相同画质下，WebP 体积比 JPG 小 30% 以上。

## 📁 结果保存
- 压缩后的图片不会覆盖原图。
- 它们会保存在原图片所在文件夹下的 **`_compressed`** 子目录中。
- 处理完成后，软件会弹窗提示。

## ❓ 常见问题
- **Q: 为什么有些图片无法压缩到 150KB 以下？**
    - A: 如果图片分辨率非常大（如 4000x3000）且包含丰富细节，即使画质降到最低也可能超过 150KB。此时建议勾选“限制最大宽度”至 1280px 或 1080px。
- **Q: 为什么 PNG 图片变大了？**
    - A: PNG 是无损格式。如果原图压缩率已经很高，再次保存并不一定能变小。建议勾选“转换为 WebP”以获得更好效果。

## 🔧 GitHub 部署与环境配置 (For Developers)

如果您从 GitHub 下载此项目，请注意以下事项：

1.  **Python 环境要求**:
    - 需要 Python 3.8+。
    - 如果您的 Python 没有添加到系统环境变量(PATH)，可能会导致脚本无法启动。

2.  **配置启动脚本 (`.bat`)**:
    - 当前的 `启动压缩工具.bat` 为了方便本地使用，可能指定了固定的 Python 路径。
    - **他人使用时**: 请右键编辑该 `.bat` 文件，检查并修改 `PYTHON_EXE` 变量为您本机的 Python 路径（或者如果已配置环境变量，可以直接修改为 `python`）。

3.  **手动安装依赖**:
    - 如果自动安装失败，请在目录下打开终端运行：
      ```bash
      pip install -r requirements.txt
      ```
